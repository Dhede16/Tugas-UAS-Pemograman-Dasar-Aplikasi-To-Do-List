## 📌 Tentang Proyek

Aplikasi **To-Do List** ini merupakan aplikasi web sederhana yang memungkinkan pengguna untuk mengelola daftar tugas sehari-hari secara efisien. Pengguna dapat menambahkan tugas baru, menandai tugas yang telah selesai, mengatur tingkat prioritas, serta menghapus tugas yang tidak diperlukan.

Proyek ini dibangun sebagai bagian dari pemenuhan **Tugas Akhir Semester (UAS)** mata kuliah **Pemrograman Dasar**.

---

## ✨ Fitur

| Fitur | Deskripsi |
|-------|-----------|
| ➕ **Tambah Tugas** | Menambahkan tugas baru dengan nama dan tingkat prioritas |
| 🎯 **Prioritas Tugas** | Tiga level prioritas: 🔴 Tinggi, 🟡 Sedang, 🟢 Rendah |
| ✅ **Toggle Status** | Menandai tugas sebagai selesai atau belum selesai |
| 🗑️ **Hapus Tugas** | Menghapus tugas satu per satu atau semua tugas yang telah selesai |
| 🔍 **Filter Tugas** | Menampilkan tugas berdasarkan status: Semua, Selesai, atau Belum Selesai |
| 📊 **Statistik** | Menampilkan jumlah total tugas, tugas selesai, dan tugas yang belum selesai |
| 🕐 **Timestamp** | Mencatat waktu pembuatan setiap tugas secara otomatis |

## 🛠️ Teknologi

- **Bahasa Pemrograman:** Python 3.8+
- **Framework Web:** [Streamlit](https://streamlit.io/)
- **Library:** `datetime` (bawaan Python)

## 🚀 Instalasi & Penggunaan

### Prasyarat

Pastikan Python sudah terinstall di komputer kamu. Cek dengan perintah:

```bash
python --version
```

### Langkah Instalasi

**1. Clone repository ini**

```bash
git clone https://github.com/Dhede16/Tugas-UAS-Pemograman-Dasar-Aplikasi-To-Do-List.git
cd Tugas-UAS-Pemograman-Dasar-Aplikasi-To-Do-List
```

**2. Install dependensi**

```bash
pip install -r requirements.txt
```

**3. Jalankan aplikasi**

```bash
streamlit run aplikasi_to_do_list.py
```

**4. Buka browser**

Aplikasi akan otomatis terbuka di browser pada alamat:
```
http://localhost:8501
```
